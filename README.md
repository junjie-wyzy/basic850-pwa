# Ogden Basic English 850

这是基于原始 Ogden 850 学习网页做的个人自用改进版。

## 改动

- 保留原网页的核心意象、例句、音标、发音、分类、卡片练习和排版。
- 每个单词卡片增加“已学会”按钮。
- 点击“已学会”后，该单词会在词表和卡片练习里一起隐藏。
- 顶部显示“今日学会”和“今日用时”。
- “恢复”按钮会恢复最近隐藏的单词，可连续点击恢复多个。

所有学习进度都保存在当前设备浏览器本地，不会上传到服务器。换手机或换浏览器后，进度不会自动同步。

## iPhone 使用

打开 GitHub Pages 地址后，用 Safari 点击：

```text
分享 > 添加到主屏幕
```

之后就可以像 App 一样从桌面打开。

## 学习建议

1. 每天先用卡片模式快速过一遍，只把真正已经熟悉的词点“已学会”。
2. 不急着一次背完 850 个词，优先把“核心意象”看懂，再看英文例句。
3. 对英语基础弱的人，建议每天固定 20 到 30 分钟，比一次学几个小时更稳。
4. 不会的词不要硬点“已学会”，让它留在页面里，第二天继续见面。
5. 读例句时先跟读英文，再看中文解释，最后用自己的话说出这个词的大概感觉。
6. 如果误点了“已学会”，点顶部“恢复”即可找回最近隐藏的词。
